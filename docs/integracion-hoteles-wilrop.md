# Propuesta de Integracion Hotelera

## WILROP Colombia Travel

### Presentacion

En WILROP Colombia Travel estamos ampliando nuestra red de aliados hoteleros para conectar su oferta con nuestra plataforma digital de comercializacion y reservas.

Nuestro objetivo es facilitar que sus habitaciones, tarifas y disponibilidad puedan integrarse de forma clara, segura y comercialmente efectiva dentro de nuestro ecosistema.

Esta iniciativa es liderada por Wilson Bautista, Fundador y CEO de WILROP, como parte del crecimiento de alianzas estrategicas con hoteles en Colombia y la region.

---

## Que buscamos integrar

Buscamos una de estas dos modalidades:

### 1. Integracion comercial basica

Ideal cuando el hotel ya cuenta con un motor de reservas propio.

En este caso necesitamos:

- URL directa del motor de reservas
- Parametros que acepta el enlace para precargar busqueda
- Nombre del proveedor del motor de reservas
- Politicas comerciales y condiciones generales

Con esta opcion, WILROP puede enviar trafico calificado directamente al motor del hotel.

### 2. Integracion completa en tiempo real

Ideal cuando el hotel desea mostrar disponibilidad real, tarifas y condiciones actualizadas dentro de nuestra plataforma.

En este caso necesitamos:

- Nombre del PMS, channel manager o booking engine
- Documentacion tecnica de la API
- Credenciales de acceso o entorno de pruebas
- Endpoints de disponibilidad, tarifas, restricciones y reservas
- Webhooks o mecanismo de notificacion de cambios, si existe
- Mapeo de habitaciones, planes tarifarios y politicas de cancelacion

Con esta opcion, WILROP puede construir una experiencia mas fluida, con mayor conversion y menor friccion para el cliente final.

---

## Beneficios para el hotel

- Mayor visibilidad comercial dentro del canal digital de WILROP
- Captacion de nuevas reservas y oportunidades B2C y B2B
- Integracion con un flujo de venta mas moderno y centralizado
- Posibilidad de escalar a integraciones mas profundas segun el proveedor tecnologico
- Mejor experiencia para el cliente al consultar y reservar

---

## Informacion minima que solicitamos

Para evaluar la integracion, por favor compartir:

- Nombre del hotel
- Ciudad y pais
- Nombre del proveedor del motor o sistema de reservas
- Enlace actual del motor de reservas
- Contacto tecnico o comercial del proveedor
- Si cuentan con API o documentacion tecnica disponible
- Si manejan parametros por fecha, ocupacion, idioma, moneda o codigo promocional

---

## Siguiente paso

Una vez recibamos la informacion, nuestro equipo revisara la viabilidad tecnica y comercial para definir la forma mas eficiente de integracion.

Si el proveedor no cuenta con API, igualmente podemos evaluar una integracion por enlace profundo al motor de reservas.
